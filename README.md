# Aetheris - Premium Minimalist Music Player

Aetheris is a sophisticated, minimalist web-based music player designed with a premium, organic aesthetic. It features high-contrast visuals, smooth transitions, and a curated sensory experience.

## ✨ Features

- **Premium UI**: A sleek, high-contrast player card set against a deep, sophisticated gradient background.
- **Dynamic Vinyl Interaction**: Rotating album art with circular progress tracking and interactive 3D tilt effects.
- **Internal Playlist Drawer**: A slide-out collection drawer integrated directly into the player interface for seamless track selection.
- **Intuitive Controls**: Precision-engineered playback controls including shuffle, repeat, and a custom-styled linear progress bar with seeking capabilities.
- **Volume Precision**: An elegant volume slider for fine-tuned audio control.
- **Autoplay**: Seamless transitions between tracks for an uninterrupted listening experience.
- **Responsive Design**: Fully optimized for both desktop and mobile viewing.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure and SVG-based circular progress.
- **Vanilla CSS3**: Custom design system using CSS variables, Flexbox, and CSS Grid.
- **JavaScript (ES6+)**: Custom audio logic, DOM manipulation, and interactive animations.

## 🚀 Getting Started

1.  **Clone/Download**: Download the project files to your local machine.
2.  **Add Your Music**:
    -   Purchase and download `.mp3` files for your favorite tracks.
    -   Place them in the project folder.
    -   Ensure the file names match the paths defined in `script.js` (e.g., `the_hills.mp3`, `starboy.mp3`).
3.  **Add Your Art**:
    -   Place your album cover images in the project folder.
    -   The code is pre-configured to look for `the hills.jpg`, `starboy.jpg`, `after dark.jpg`, and `kaleo.jpg`.
4.  **Launch**: Open `index.html` in any modern web browser to explore the collection.

## 🎨 Design Philosophy

Aetheris is built on the principles of **clarity and separation**. By isolating the interactive elements within a clean white card and utilizing high-contrast charcoal accents, the interface remains functional yet breathtakingly simple.
